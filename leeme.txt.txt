Sistema de votacion Matias Gutierrez

El sistema fue creado y ejecutado en ambiente local mediante el software XAMPP,
con su sistema de gestion de base de datos MySQL y servidor web Apache.

-- Configuraciones de ejecucion:

1- Abrir carpeta en SuDisco:\xampp\htdocs
2- Abrir git bash dento de la carpeta htdocs
3- Ingresar el comando: git clone https://github.com/Matiasgutierrezdu/Sistema-Votacion.git
4- Visualizar la correcta creacion de carpetas
5- Abrir mediante editor de codigo la carpeta creada
6- Dar inicio a Apache y MySQL en Xampp

-- Para importar base de datos ---

7- Ingresar a la direccion: localhost, ingresar a phpMyAdmin
8- Crear nueva base de datos
9- Ir a Importar desde carpeta sql archivo de base de datos: db.sql luego continuar.

-- Ingreso sistema --

10- Ingresar en el navegador el siguiente enlace: http://localhost/sistemavotacion/
